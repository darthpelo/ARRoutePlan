ARRoutePlan
===========

Simple excercise to test:
* autocomplete search
* calendar picker

UI ispiration by Kayak app.
At the moment this app supports **only iOS 7**, but I'm not use iOS 7 only methods or frameworks,
so porting to iOS 6 is very simple .
